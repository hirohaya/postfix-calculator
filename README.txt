Programa feito em Python 3.4.2 para a matéria Laboratório de Programação II ministrada pelo professor
Alfredo Goldman no Instituto de Matemática e Estatística da USP.

Execução:

	1. Execute o programa sem argumentos na linha de comando utilizando Python 3 (ex: py -3 miniep5.py)
	2. O programa irá pedir para você entrar com a expressão em notação posfixa e
	   imprimirá o resultado logo em seguida.
